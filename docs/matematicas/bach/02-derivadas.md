# 1. Límites

!!! note "Definición"
    Decimos que \\(\\lim_{x\\to a} f(x)=L\\) si los valores de \\(f(x)\\) pueden hacerse tan cercanos como queramos a \\(L\\) cuando \\(x\\) se aproxima a \\(a\\).

!!! example "Ejemplo"
    \\[
    \\lim_{x\\to 2} (3x^2-5x+1) = 3\\cdot2^2 - 5\\cdot2 + 1 = 3
    \\]
